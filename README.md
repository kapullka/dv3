
# Dispatch Tracker v3

Features:
- Months created dynamically by calendar; weeks start on Monday.
- Add New Month button (copies employees from previous month).
- Employees copied from previous month but can be removed.
- Right panel: Employee list and Monthly Plan with Current Profit column.
- Weekly totals and monthly total.
- Basic conditional coloring/status indicator for progress vs plan.
- Months are NOT locked automatically.

How to run locally:
1. pip install -r requirements.txt
2. streamlit run app.py
